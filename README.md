# ML Micro-Projects

A collection of Python tools for data analysis, numerical computing, and machine learning workflows.

## Projects

### Data Summary Tool
Exploratory data analysis utility. Computes descriptive statistics and visualizes distributions for tabular data.
- **Stack:** Python, Pandas, Matplotlib

### NumPy Vector Explorer
Numerical computing toolkit. Demonstrates array creation, NaN-aware statistics, and vectorized operations.
- **Stack:** Python, NumPy
